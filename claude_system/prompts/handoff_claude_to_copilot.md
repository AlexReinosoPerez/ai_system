# Prompt — Claude → Copilot Handoff

## Purpose

This text represents an explicit transfer of execution authority
from Claude to Copilot.

Without this text, Copilot must not be used.

---

## Mandatory Phrase

Claude must emit the following phrase verbatim:

PLAN APPROVED. Proceed with Copilot.

---

## Meaning

By emitting this phrase, Claude asserts that:
- a plan exists
- the plan is complete
- no architectural uncertainty remains
- execution is purely mechanical

After this phrase:
- Claude must stop implementing
- Copilot may begin execution
