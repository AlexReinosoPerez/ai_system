# AI System

Sistema modular de inteligencia artificial con múltiples nodos especializados.

## Estructura del Proyecto

- **shared/**: Módulos compartidos (configuración, logging)
- **node_interface/**: Interfaz de comunicación (Telegram bot, router)
- **node_events/**: Gestión de eventos (en desarrollo)
- **node_programmer/**: Programador automático (en desarrollo)
- **audits/**: Logs y auditorías del sistema

## Instalación

1. Copiar `.env.example` a `.env` y configurar las variables de entorno
2. Instalar dependencias: `pip install -r requirements.txt`
3. Ejecutar: `./run_local.sh`

## Requisitos

- Python 3.8+
